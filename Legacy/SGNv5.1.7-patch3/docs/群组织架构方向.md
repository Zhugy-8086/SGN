# SGN 群组织架构方向（简述）

> **定位**：长期架构方向简述。详细设计见 `fixes_相关修复/v5.1.7_群组织架构设计/`。

## 架构层级

```
神经组层（任务复杂度驱动涌现）
神经节层（临时动态结构，复数矩阵表达 + 回溯校验）
神经元层（持久结构，竞争排名 + 赫布学习）
统一基础（时序量纲 + 物理自洽性）
```

## 核心思想

1. **归一化贯穿全程**：不是 BatchNorm，而是数据压缩 + 符号向 ID 化（二值化→磁化对比→神经节压缩→神经组归一化）
2. **群组织 = 神经组**：归一化的延续，不是额外机制。任务复杂度超出神经节能力时自然涌现
3. **专精竞争涌现**：赫布学习 + 贡献度加权的自然统计结果，不是硬编码分配
4. **统一图**：存储物理特征标签（brightness/edge_pattern/spatial_consistency），与任务无关

## 任务自由度

架构不绑定任务类型：
- 底层和统一图保持任务无关
- 所有任务逻辑在神经节/神经组层
- 支持分类、多图片识别、图片认知、序列识别等
